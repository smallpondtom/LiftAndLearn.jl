LiftAndLearn.ChafeeInfante.ChafeeInfanteModel — Type
mutable struct ChafeeInfanteModel <: LiftAndLearn.AbstractModel

\[\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2} + \lambda (u - u^3)\]

where $u$ is the state variable, $\lambda$ is the coefficient of nonlinearity.

LiftAndLearn.ChafeeInfante.finite_diff_mixed_model — Method
finite_diff_mixed_model(model::chafeeinfante, μ::Real)

Create the matrices A (linear operator), B (input operator), and E (cubic operator) for Chafee-Infante model using the mixed boundary condition. If the spatial domain is [0,1], then we assume u(0,t) to be homogeneous dirichlet boundary condition and u(1,t) to be Neumann boundary condition of some function h(t).

Arguments

  • N::Real: spatial dimension
  • Δx::Real: spatial grid size
  • μ::Real: diffusion coefficient

Returns

  • A::SparseMatrixCSC{Float64,Int}: linear operator
  • B::SparseMatrixCSC{Float64,Int}: input operator
  • E::SparseMatrixCSC{Float64,Int}: cubic operator
